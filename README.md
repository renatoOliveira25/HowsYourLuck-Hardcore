# 🎲 How's Your Luck?

Do you feel lucky today?

Every time you save a file in Visual Studio Code, fate spins the chamber.

There is a **1 in 1000 chance** that your luck runs out...

💀 A random file in your project will be deleted.

## How it works

Every save triggers a random roll.

If the unlucky number appears:

1. The chamber spins
2. A random file is selected
3. The file is deleted

## Features

- 🎲 Random luck roll on every save
- 💀 Russian roulette mechanics
- 📂 Random file deletion
- ⚡ Works with manual and auto-save

## Warning

This extension can delete files from your project.

Use it **only if you are brave enough**.

## Requirements

Works with any project in Visual Studio Code.

## Enjoy... if you dare.
